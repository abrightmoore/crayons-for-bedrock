# crayons-for-bedrock
Crayons add on for Minecraft Bedrock

Red, Yellow, Blue and Green crayons, with mixing colours:
Red + Blue = Purple... etc.

Highlighter on Purple, Green and Yellow for 'glow'.

(Drawing on stone with crayons that you can also eat)
